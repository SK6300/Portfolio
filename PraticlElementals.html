About This Sphere

This is an interactive particle sphere visualization. You can drag to rotate it using your mouse or touch.

Click the microphone button to enable voice reactivity. When enabled, the particles will react to the sound level detected by your microphone, causing a "volcanic" eruption effect and changing color.

New Feature: Use the 📁 icon to load any .glb, .gltf, or .obj 3D model. The particles will smoothly morph into the object's shape before revealing it!

Use the Pause and Run buttons, or the top-right button to control animation and the info page.

Drag this info panel to rotate it in 3D! It will slide a bit.
